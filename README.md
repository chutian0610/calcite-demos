# Apache Calcite 教程项目

本项目提供了 Apache Calcite 的入门教程和示例代码，帮助开发者快速了解和使用 Apache Calcite。

## 项目简介

Apache Calcite 是一个动态数据管理框架，提供了标准的 SQL 查询语言、查询优化和连接各种数据源的能力。本项目通过一系列教程，帮助开发者逐步掌握 Calcite 的核心功能。

## 项目结构

```
calcite-tutorials/
├── calcite-bom/           # Bill of Materials，管理依赖版本
├── calcite-util/          # 通用工具类
├── tutorials/             # 入门教程
│   ├── quickstart/     
│   ├── tutorial01/   
│   ├── ... ... 
│   └── tutorialxx/
├── pom.xml                # 项目根POM文件
└── README.md              # 项目说明文档
```
